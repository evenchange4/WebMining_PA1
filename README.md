## Run

* 查看bash 位置

```
$ which bash
=> /bin/bash
```

* 調整權限

```
$ chmod +x compile.sh
$ chmod +x execute.sh
```

* compile 助教給的Text processing toolkit

```
$ ./compile.sh
```

* 執行程式

```
$ ./execute.sh [-r] -i tmp2/queries/query-5.xml -o ans -m tmp2/model-files -d tmp2/CIRB010
```
